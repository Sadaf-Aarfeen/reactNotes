# 📝 Notes App (React + Tailwind CSS)

A simple and responsive Notes Application built using **React** and **Tailwind CSS**.
Users can add and delete notes with a clean UI and smooth user experience.

---

## 🚀 Features

* ➕ Add notes with title and details
* 🗑️ Delete notes instantly
* 📱 Fully responsive layout (mobile + desktop)
* 🎨 Modern UI 
* ✂️ Text handling (no overflow, clean wrapping)
* 🧾 Scrollable notes section

---

## 🧠 Key Learnings

* Managing state using `useState`
* Handling form inputs in React
* Dynamic rendering using `.map()`
* Deleting items using index
* Fixing layout & overflow issues
* Responsive design with Tailwind CSS
* Scroll behavior & UI improvements
